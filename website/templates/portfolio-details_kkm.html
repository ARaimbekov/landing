Информация по проекту

  • Категория: Видео-конференц связь
  • Заказчик: Красноярский краевой медицинский информационно-аналитический центр
  • Дата выполнения проекта: 01 March, 2021

Описание

Группа компаний «Интант» создала систему видеоконференцсвязи для нужд Красноярского краевого медицинского информационно-аналитического центра. Решение ВКС строилось на базе оборудования Telepresence компании Cisco Systems. В основу предложенного решения вошли такие продукты:

1. Cервер сигнализации, используемый для управления вызовами, на базе программно-аппаратного комплекса Cisco VCS Control

2. Cервер управления системой ВКС на базе программно-аппаратного комплекса Cisco TMS

3. Мост видеоконференцсвязи для организации многоточечной связи на базе Cisco MCU 4200 и устройства конечных абонентов на основе программных клиентов Cisco Movi.

Система позволяет организовывать видеоконференции с общим количеством до 30 одновременных участников, предоставляет возможность подключения в ВКС по Web, возможность использовать видео высокого разрешения, формирует индивидуальные видео-потоки для каждого абонента в зависимости от его условий и требований с возможностью транскодирования в режиме реального времени. Реализованная сисетма является многофункциональной и масштабируемой, поддерживает возможность подключения в будущем конференц-комнат и систем комбинированного назначения без модернизации ядра системы.
